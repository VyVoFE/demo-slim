# demo-slim
Tested by Postman.
Require composer install.
List API routes:
- /: welcome
- /getAll: get all records - GET
- /getOne: get one record by id - POST 
- /create: create a new record - POST - body: [name, age, color]
- /update: edit a record by id - POST
- /delete: delete a record by id - POST
